# Usage

Domeport Pro is designed for artists, producers, and broadcasters working with dome and planetarium configurations, including the [Satosphere](https://sat.qc.ca/en/satosphere/). It lets you visualise DomeMaster format immersive content in real time, whether pre-rendered or generated live in multimedia creation software.

## Input sources

Domeport Pro supports a wide range of input sources:

- **Video playback**: load standard image and video formats directly
- **NDI**: receive live video streams over a local network
- **Spout** (Windows): share textures in real time between applications
- **Syphon** (macOS): share textures in real time between applications

> NDI, Spout, and Syphon are only available in the standalone version. The [web version](https://dev.domeport.sat.qc.ca/) supports file-based content only.

## Projection formats

Select your projection format from the interface:

- **DomeMaster**: circular fisheye projection for full-dome screens
- **Equirectangular**: 360° panoramic format for dome or VR previewing

## Playback controls

When a video file is loaded:

- **Play / Pause**: start and stop playback
- **Scrubber**: drag to jump to any position in the video

## Camera controls

Navigate the 3D dome view with the built-in camera:

- **Fly mode**: free-fly navigation through the scene: use `W` `A` `S` `D` to move and the mouse to look around
- **Walk mode**: ground-level navigation: `W` `A` `S` `D` moves along the floor plane
- **Aperture**: set the dome coverage angle (180° for a full hemisphere, 210° for an extended dome)

## Use cases

Connect to **[TouchDesigner](https://derivative.ca/)**, **[ossia score](https://ossia.io/)**, **[Hydra](https://hydra.ojack.xyz/)**, **[Isadora](https://troikatronix.com/)**, or any NDI-compatible software to stream live content directly into Domeport Pro.
